# FedCiv Lab CLI

## Download
https://github.com/sambyers/fedciv-lab-cli/raw/main/dist/civlab

## Setup
```shell
export CIVLAB_URL=http://x.x.x.x
chmod +x civlab
```

## Usage
```shell
civlab --help
civlab status -name DEVICE_NAME
```
